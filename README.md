# 🏰 Пульт Холопа

Панель-помощник для игры **@holop** в Telegram: набеги, роли холопов, будильники,
пещеры, разведка. Работает на твоём аккаунте, у тебя на компьютере.

---

## 🪟 Установка на Windows

### Шаг 1. Скачай папку
Нажми зелёную кнопку **`Code`** вверху страницы → **`Download ZIP`**.
Распакуй архив (правой кнопкой → «Извлечь всё»). **Из архива ничего не запускай — сначала распакуй.**

### Шаг 2. Python (нужен один раз)
Если Python ещё не стоит:
1. Открой https://www.python.org/downloads/
2. Скачай и запусти установщик.
3. ⚠️ **Внизу первого окна поставь галочку `Add python.exe to PATH`** — это обязательно.
4. Нажми `Install Now`, дождись конца.

> Если ставил Python «из Microsoft Store» — лучше переустанови с сайта выше.
> Стоковая заглушка из Store часто не работает как надо.

### Шаг 3. Запуск
Дважды кликни **`START.bat`**.
- Первый запуск сам поставит нужную библиотеку (~минуту, видно в окне).
- Откроется чёрное окно — **не закрывай его**, пока оно открыто, пульт работает (свернуть можно).
- Сам откроется браузер. Если нет — открой вручную: http://127.0.0.1:8777/

### Шаг 4. Вход
На странице: номер телефона → код из Telegram → (пароль 2FA, если стоит). Готово.

## 🍎 macOS
Дважды кликни **`START-mac.command`** (первый раз может понадобиться правый клик → «Открыть»).

---

## 🆘 Если не работает — пришли логи

В папке рядом с `START.bat` появляются два файла:
- **`startup_log.txt`** — что происходило при запуске (нашёлся ли Python, установка библиотеки).
- **`hub_error.log`** — полный текст ошибки, если пульт упал.

**Просто пришли эти два файла** — по ним ошибку видно и можно быстро починить.
(В них нет твоего пароля/сессии — только техническая информация.)

## 🔒 Безопасность и риски
- Твой вход хранится **только у тебя** в файле `config.json` в этой папке. Никуда не уходит.
  **Никому его не пересылай** — это ключ от твоего Telegram-аккаунта. (В репозиторий он не попадает — см. `.gitignore`.)
- Это **самобот**: действует от твоего личного аккаунта, что против правил Telegram.
  За автоматизацию аккаунт теоретически могут ограничить или забанить. Используешь на свой риск.
