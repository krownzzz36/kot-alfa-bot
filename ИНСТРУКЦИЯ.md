# 📖 Пульт Холопа — полная инструкция

Помощник для игры **@holop** в Telegram. Работает на твоём аккаунте, у тебя на
компьютере. Ставится один раз, дальше — просто запускаешь.

> ✅ **Проверь версию.** В шапке пульта справа от «🏰 Холоп — Пульт» есть номер
> версии (например `v2026.07.21-4`). Если её нет или она старее — у тебя устаревшая
> сборка, скачай свежую (см. раздел «Обновление»).

## Установка (Windows)

### Шаг 1. Скачай папку
На странице проекта → зелёная кнопка **`Code`** → **`Download ZIP`**.
Скачается архив. **Распакуй его** (правой кнопкой → «Извлечь всё» → «Извлечь»).
👉 Запускать нужно **из распакованной папки**, а не изнутри архива.

### Шаг 2. Установи Python (один раз)
Если Python ещё не стоит:
1. Открой https://www.python.org/downloads/
2. Скачай и запусти установщик.
3. ⚠️ **Внизу первого окна поставь галочку `Add python.exe to PATH`.** Это обязательно.
4. Нажми `Install Now`, дождись конца.

> Не ставь Python «из Microsoft Store» — часто не работает как надо. Ставь с сайта выше.

### Шаг 3. Запусти
Дважды кликни **`START.bat`**.
- Первый запуск сам поставит нужную библиотеку (~минуту, видно в окне).
- Откроется чёрное окно — **не закрывай его** (свернуть можно). Пока оно открыто — пульт работает.
- Сам откроется браузер со страницей пульта. Если нет — открой вручную: **http://127.0.0.1:8777/**

> **Выскочило синее окно «Windows защитила ваш компьютер»?** Это перестраховка на
> скачанных файлах, не ошибка. Нажми **`Подробнее`** → **`Выполнить в любом случае`**.

## Установка (Mac)
Дважды кликни **`START-mac.command`**. Первый раз может понадобиться правый клик → «Открыть».
Дальше — как на Windows (браузер откроется сам).

---

## Первый вход
На странице пульта:
1. Введи **номер телефона** (как в Telegram, с `+7…`).
2. В Telegram придёт **код** — введи его.
3. Если стоит **облачный пароль (2FA)** — введи и его.

Готово — откроется сам пульт. Вход сохраняется в файл `config.json` в папке
и больше спрашиваться не будет.

## Как понять, что всё работает
- В шапке — номер версии (значит пульт свежий).
- Чёрное окно (Windows) открыто и не закрылось — сервер жив.
- На вкладке с зелёной плашкой **🟢 Работает** — процесс запущен.
  ⚪ «Не запущен / Остановлен» — процесс стоит.
- В большом чёрном поле бежит лог — это «живая» картинка того, что делает бот.

## Вкладки

### ⚔️ Набеги — главное
Авто-бой по твоему списку целей по кругу + защита от бочки (динамита).

**Как поставить цели (важно!):**
1. Справа поле **«🎯 Цели»** — впиши ников соперников, **по одному в строке**.
2. Нажми **«💾 Сохранить список»**. ← без этого шага цели не применятся!
3. Нажми **«▶ Запустить»**.

> ⚠️ Если запустить набеги **с пустым списком** — бот **никого не бьёт** и просто ждёт,
> пока ты добавишь цель. Он НЕ выбирает цели сам. Ников берёшь либо руками, либо со
> вкладки «🔎 Найти цели».

- **▶ Запустить / ⏹ Остановить** — старт и стоп цикла.
- **🌙 Ночной режим** — для фарма на ночь (на Mac ещё и не даёт компу уснуть и
  перезапускает бота, если завис). На Windows просто держит набеги включёнными —
  не закрывай окно.
- Список целей можно править **на лету**: поменял → «Сохранить список» → бот
  подхватит в течение минуты, останавливать не нужно.

### 🎭 Роли холопов
Перегон холопа в нужную профессию (бот крутит «выгнать → захватить», пока не выпадет нужная).
1. Впиши **ников холопов**, по одному в строке.
2. Выбери **профессию**: Воин, Ополченец, Пахарь, Волхв, Ремесленник, Зодчий, Лазутчик.
3. Кнопки:
   - **▶ Перегнать** — начать.
   - **🔍 Проверить** — просто глянуть текущую роль (первого из списка), ничего не меняя.
   - **Холостой (dry-run)** — прогон вхолостую, без реальных действий (проверить, что всё живо).

### ⏰ Будильники
Отложенные «мигалки» в чат к моменту, когда у холопов спадёт охрана.
1. Укажи **за сколько минут до спадения щита** предупреждать (по умолчанию 5).
2. Кнопки: **⏰ Поставить будильники** / **🧹 Снять все** / **Холостой**.

### 🕳️ Пещеры
Автопроход трёх пещер по очереди до стоп-уровня.
Параметры:
- **Мин. шанс победы, %** (по умолчанию 85) — ниже этого шанса бот не лезет в бой.
- **Мин. HP** (20) — при HP ниже выходит, не рискует.
- **Стоп-уровень** (10) — на каком уровне пещеры остановиться.
- **Только пещера** (необязательно) — впиши кусок названия, если нужна одна конкретная.
Кнопки: **🕳️ Пройти пещеры** / **Холостой**.

### 🔎 Найти цели
Сканирует богатых бьющихся соперников (сортировка по серебру) и выдаёт список ников.
1. **Сколько целей найти** (по умолчанию 10) и **сколько страниц сканировать** (6).
2. Нажми **🔎 Найти цели** — в логе появится список ников.
3. **Скопируй ников** из лога и вставь в «⚔️ Набеги» → «🎯 Цели» → «Сохранить список».

> Это только поиск и список — сам он никого не бьёт.

### 📊 Разведка (КД/щиты)
Показывает, когда у каждого соперника спадёт щит/КД и восстановится HP (время по МСК).
1. Впиши **ников для проверки**, по одному в строке.
2. **🔎 Проверить** — показать состояние.
3. **⏰ Проверить + напоминалки** — ещё и поставит напоминания «за N минут: готовься к атаке на X»
   (N — поле «Напомнить за … минут»).
4. **🧹 Снять напоминалки** — убрать их.

## Значки состояний в логе
Рядом с ником в набегах/разведке видишь короткий статус:

| Значок | Что значит |
|---|---|
| 🏆 победа | цель побита успешно |
| ⌛ КД … | цель на перезарядке (нельзя бить), указано, сколько осталось |
| 🛡️ щит … | цель под защитным щитом |
| 💤 реген … | ждём, пока цель восстановит HP |
| 🧱 пробиваю | ломаю частокол/ров у цели |
| ❌ снят (скамейка) | проиграл этой цели — временно отложил её до конца сессии |
| 🚫 недоступен | свой клан или цель ниже уровня — бить нельзя |
| 💤 слаб | цель слишком слабая |

## Частые проблемы

**«Python не найден»**
→ Не поставил галочку «Add python.exe to PATH» при установке Python. Переустанови
Python с галочкой (см. Установка, шаг 2) и запусти START.bat снова.

**Окно мигнуло и сразу закрылось**
→ Запускал прямо из архива. Сначала распакуй ZIP в обычную папку и запускай оттуда.

**Синее окно «Windows защитила ваш компьютер»**
→ «Подробнее» → «Выполнить в любом случае». Это не ошибка.

**Кнопка нажимается, но «серебро не ищет» / странности в логе**
→ Скорее всего у тебя **старая версия**. Проверь номер версии в шапке; если старый —
скачай свежую (см. Обновление).

**Набеги идут не по тем целям**
→ Проверь список в «🎯 Цели» и что ты нажал «💾 Сохранить список». Пустой список =
бот никого не бьёт.

**Ничего не помогло — пришли логи.**
В папке рядом со `START.bat` появляются файлы **`startup_log.txt`** и **`hub_error.log`**.
Пришли их — по ним видно точную причину. (Пароля/сессии в них нет, только техника.)

## Обновление
Баги чинятся в общем репозитории. Чтобы получить исправления:
1. Скачай свежий ZIP заново (**Code → Download ZIP**), распакуй в **новую** папку.
2. Запусти `START.bat`, войди заново (30 секунд).
   *(Или скопируй свой файл `config.json` из старой папки в новую — тогда вход сохранится.)*
3. Проверь: номер версии в шапке стал новее. Старую папку удали, чтоб не путаться.

---

## Безопасность и риски
- Твой вход хранится **только у тебя** в файле `config.json` в папке. Никуда не уходит.
  **Никому этот файл не пересылай** — это ключ от твоего Telegram-аккаунта.
- Это **самобот**: действует от твоего личного аккаунта, что против правил Telegram.
  За автоматизацию аккаунт теоретически могут ограничить или забанить. Используешь на свой риск.
- Не ставь безумных скоростей и не молоти сутками без меры — так меньше шансов словить ограничение.
